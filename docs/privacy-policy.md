# Privacy Policy for Focus Cat Lite

Focus Cat Lite is designed to be local-only and privacy-friendly.

## What the app does

- Lets users start a focus session
- Lets users manually record distractions
- Shows session duration and distraction totals

## Data collection

Focus Cat Lite does not collect personal data.
Focus Cat Lite does not send data to a server.
Focus Cat Lite does not use analytics, ads, login, or a backend.

## Permissions

The app does not require special tracking permissions.
The app does not require Accessibility Service, UsageStats, or Device Admin.

## Local storage

The current Lite release keeps the experience simple and local-first. Any future local storage features should stay on-device unless the user explicitly needs something else.

## Contact

If you have questions about privacy, contact the app owner through the project or store listing support contact.
